# shortcut key

  1. 注释单行：Ctrl+/
  2. 注释多行：Ctrl+Shift+/
  3. 转换大写：Ctrl+K+U
  4. 转换小写：Ctrl+K+L
  5. 撤销：Ctrl+Z
  6. 恢复撤销：Ctrl+Y
  7. 就近选择相同项：Ctrl+D
  8. 选择当前文件所有匹配项：Alt+F3
  9. 向上扩展一层：Ctrl+Shift+A
  10. 整行的上下移动：Ctrl+Shift+↑ 或 Ctrl+Shift+↓
  11. 复制行或选中项：Ctrl+Shift+D
